منصة حصيف – النظام التشغيلي الوطني للتكامل التنموي

هذه النسخة الأمامية (Front-End) تم إعدادها لاستخدامها كنموذج عمل تشغيلي قابل للتطوير، يوضح شكل الحسابات والخدمات الأساسية Core Services لكل قطاع، استناداً إلى الملف التشغيلي والتقني الشامل لمنصة حصيف.

المكوّنات الأساسية Core Modules (ملخص)

يمكن ربطها لاحقاً بـ DB و APIs
الحلول Solutions

استقبال وطرح الحلول التنموية من الأفراد والجهات لمعالجة تحديات محددة.

المشاريع Projects

تحويل الحلول إلى مشاريع تنفيذية بميزانيات ومراحل وشركاء.

الفعاليات Events

إدارة الملتقيات، المؤتمرات، ورش العمل، الحملات التوعوية.

المرافق Facilities

قائمة بالمرافق والقاعات والمراكز المتاحة للأنشطة والفعاليات.

التطوع Volunteering

نظام موحد لإدارة المتطوعين والفرص وساعات التطوع.

التوظيف Employment

الوظائف والعقود المؤقتة المرتبطة بالمبادرات والمشاريع.

التمويل Funding

طلبات المنح والقروض التنموية وتمويل المشاريع.

الرعايات Sponsorship

رعايات تجارية ومجتمعية للفعاليات والمشاريع.

المحفظة المالية Wallet

سجل التدفقات المالية والمنح والدفعات والاستثمارات.

الشراكات الاستراتيجية المقترحة

نماذج مقترحة للجهات التي يمكن أن تكون شركاء رئيسيين للمنصة
جهة حكومية
وزارة الاتصالات وتقنية المعلومات

شريك استراتيجي لتمكين البنية الرقمية للمنصة وربطها بالبرامج الوطنية للتحول الرقمي.

  • دعم البنية التحتية التقنية.
  • تيسير الربط مع الجهات الحكومية ذات العلاقة.
  • تمكين المنصة ضمن منظومة الحلول الرقمية الوطنية.
مركز وطني
المركز الوطني لتنمية القطاع غير الربحي

شريك رئيسي لتعظيم أثر القطاع غير الربحي عبر المنصة ورفع كفاءة التمويل والحوكمة.

  • اعتماد نماذج العمل للحساب غير الربحي.
  • تشجيع الجمعيات على استخدام المنصة.
  • ربط تقارير الأثر بأهداف تطوير القطاع غير الربحي.
جهة تمكين الأعمال
منشآت – الهيئة العامة للمنشآت الصغيرة والمتوسطة

شريك لتمكين ريادة الأعمال وربط مشاريع رواد الأعمال ببرامج التمويل والحاضنات.

  • دعم برامج الحاضنات والمسرّعات.
  • نشر الفرص لرواد الأعمال عبر المنصة.
  • ربط بيانات المشاريع بالمؤشرات الوطنية.

  • ملف شخصي مهني وتطوعي.
  • الربط مع منصات التوظيف والتدريب.
  • متابعة الطلبات والقبولات.

  • لوحات تحكم للمشاريع والبرامج.
  • إدارة المتطوعين والفرص.
  • ربط مباشر بالمانحين والجهات الحكومية.

  • تكامل API مع أنظمة الجهة.
  • تقارير متقدمة ومسارات حوكمة مخصصة.
  • مسارات شراكات واستثمار وتنمية مجتمعية.
